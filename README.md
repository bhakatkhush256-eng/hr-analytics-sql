Title:
HR Analytics — SQL Project

Description:
This project focuses on analyzing employee data using SQL to uncover insights related to workforce distribution, compensation, performance, and attrition. It demonstrates the use of SQL for real-world HR analytics scenarios.

Tools & Techniques:

SQL
Joins
Aggregations
Subqueries
CASE Statements
Window Functions

Datasets:

departments — Contains department names and locations
employees — Includes employee details, salary, and employment status
performance — Stores annual performance ratings

Key Analyses:

Total employees by department
Average salary by department
Gender distribution across the organization
Employee attrition rate
Top 5 highest-paid employees
Top-performing employees
Employees earning above company average salary
Salary ranking within departments
Years of experience analysis
Comprehensive employee summary report
